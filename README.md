# Command JSON Addon for Patrimeo

A Patrimeo addon that fetches asset transactions by executing a command-line script that outputs JSON.

## Installation

```bash
composer require patrimeo/addon-command-json
```

## Usage

Configure the addon with a command that outputs JSON in the Patrimeo transaction format. The command's output will be parsed and converted into asset transactions.

## Requirements

- PHP 8.4+
- Laravel 11+ or 12+
- Filament 4.0+
